Pacotes do NuGet necessários:

1º System.DataSqlClient por Microsoft
2º iTextSharp por bruno lowagie
3° BCrypt.NET por ChrisMckee

Ver arquivo SQL para criar banco de dados e inserir usuário admin.

Usando o sistema.

1º Inserir operações de costura e seus devidos valores;
2º Inserir um cliente;

3º Para montar um custo, selecionar a operação da grid direita, ao carregar nas text box, escolher a fase colocar uma frequência se necessário, e incluir na grid, então é só salvar;
Deve ser preenchido os dados referentes ao modelo;

4º No menu modelos, escolher um modelo, clicar em detalhes, será detalhado as operações do modelo, para gerar uma cobrança basta clicar em GERAR COBRANÇA, na nova janela preencher a grid da direita com auxílio da grid esquerda, pode ser acrescentado um percentual sobre o custo do processo, inserir todas as operações do modelo na grid direita. Ao final colocar o número do talão, o tipo do produto e salvar;

5º Para criar usuários basta acessar o menu usuários e criá-lo, alterar ou excluir um usuário na mesma janela. Usuários em situação "USER" não podem acessar o menu de usuários;

6º É necessário criar uma pasta no disco local C: chamada pdf, para criar e exibir os arquivos pdf;
